# Prueba Técnica: Sistema de Gestión de Tareas

Este proyecto contiene una aplicación web fullstack para la gestión de tareas, con frontend en Next.js y backend en Nest.js.

## Estructura
- `frontend/`: Aplicación Next.js
- `backend/`: API Nest.js
